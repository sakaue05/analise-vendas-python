# Análise de Vendas com Python

Projeto de análise de dados utilizando Python, Pandas e Matplotlib.

## Tecnologias utilizadas

* Python
* Pandas
* Matplotlib

## Objetivo

Explorar um dataset de vendas e gerar insights sobre faturamento, produtos e cidades com maior desempenho.

## Análises realizadas

* Cálculo de faturamento total
* Faturamento por produto
* Faturamento por cidade
* Ticket médio
* Identificação da maior venda

## Visualizações

O projeto gera gráficos para facilitar a interpretação dos dados:

* Faturamento por produto
* Faturamento por cidade
* Quantidade vendida por produto
